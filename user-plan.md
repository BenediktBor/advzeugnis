# User Plan

There are two types of user accounts: Solo and School.

Solo users are treated as if they are the admin user of a school with only a single user.
School users have different privilege levels:

-   admin: can do anything. Add/remove teachers, edit template sentences etc.
-   editor: can only edit template sentences
-   teacher: can only create local students and create their certificate text ("Vorlage") hidden

New nav element: School/User depending on user type. School provides overview of all active teachers and their roles.
